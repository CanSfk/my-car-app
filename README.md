# My-Car

## 🔹 Proje Hakkında
My-Car uygulaması, kullanıcıların araç listesini incelemesine ve her araç hakkında yapılan kullanıcı yorumlarını görmesine olanak tanır. Uygulama ile araçları favorilere ekleyebilir, detaylı açıklamalarını görüntüleyebilir ve arama filtreleri ile istediğiniz özelliklere uygun araçları hızlıca bulabilirsiniz.

## 🔹 Kullanılan Flutter Sürümü
- Flutter 3.41.x 

## 🔹 Kullanılan API
Araç verileri için şu API kullanılmıştır:  
[https://myfakeapi.com/api/cars](https://myfakeapi.com/api/cars)

## 🔹 Çalıştırma Adımları
1. Projeyi klonla:
```bash
git clone https://github.com/CanSfk/my-car-app.git
cd my-car-app
```

2. Gerekli paketleri yükle:
```bash
flutter pub get
```

3. Uygulamayı çalıştır:
- Android Emulator veya gerçek cihazda:
```bash
flutter run
```

4. Favoriler, detay sayfası ve araç açıklamaları uygulama içinde test edilebilir.

## 🔹 Özellikler
- Araç listesi ve detay sayfaları  
- Favori araç ekleme ve çıkarma  
- Araç açıklamaları ve kullanıcı yorumları  
- Arama ve filtreleme
